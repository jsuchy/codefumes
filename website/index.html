Official CodeFumes.com Ruby Gems

CodeFumes.com is a site intended to simplify the process of gathering metrics and associating them with a particular revision of code. The gems described below integrate with this website and are intended to simplify the process of "fleshing out" the pictures you want to draw around your codebase.

Gems

    • codefumes
    • [wiki]
    • [rdoc]
    • [repo]
    • [bugs/requests]
    • [mailing list]

    The 'codefumes' gem is an implementation of the CodeFumes.com API. The intention of the gem is to simplify integration with CodeFumes.com for developers of other libraries & and applications.

    • codefumes_harvester
    • [wiki]
    • [rdoc]
    • [repo]
    • [bugs/requests]
    • [mailing list]

    The codefumes_harvester gem provides a set of high-level tools for gathering history and common metrics from a local repository and sending them to CodeFumes.com.

Contributing

Improvements in the code and/or documentation are always welcome. The gems each have a wiki on the Github page (see gem links above). You can contribute to the code by forking a repository and sending a pull request to Tom Kersten (github handle: tomkersten).

Community Discussion/Mailing Lists

General discussion around CodeFumes.com and associated tools/content is welcome on the codefumes Google Group

Bug reports, Feature Requests

Feel free to use the "Issues" area of the Github repository for this project. The link for each gem is provided above.
